# UI設計

**作成日：** 2026-04-26  
**ステータス：** 確定

---

## デザインコンセプト

「家族の中心で会話のきっかけになる、みんなを輪で繋ぐ」

柴犬を中心に家族メンバーが輪になるビジュアルで「みんなで見守っている」感を表現する。
タイムラインの担当者色と FamilyCircle の色を統一し、誰の記録かを直感的に把握できるようにする。

---

## テーマカラー

v0.dev で生成したパレットをベースにしている（globals.css の CSS カスタムプロパティで管理）。

| トークン | 用途 | イメージ |
|---|---|---|
| `--primary` | メインカラー・ボタン・アイコン | 柴犬の赤茶色 |
| `--accent` | サブアクション・グラデーション | やや明るい暖色 |
| `--background` | 画面背景 | 縁側のポカポカ感・クリーム系 |
| `--card` | カード背景 | オフホワイト |
| `--warning` | 排泄アラート警告 | オレンジ系 |
| `--destructive` | 削除アクション | 落ち着いた赤 |

### 家族メンバーカラー（アバター色）

担当者を色で識別するために固定の色を割り当てる。FamilyCircle とタイムラインカードで共通。

| メンバー | Tailwind クラス |
|---|---|
| お母さん | `bg-rose-300` |
| お父さん | `bg-sky-300` |
| お姉ちゃん | `bg-amber-300` |
| お兄ちゃん | `bg-emerald-300` |
